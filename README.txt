--Описание программы:
-Программа представляет собой консольное Java-приложение для обработки файлов с расширением .sb, содержащих данные сотрудников.

--Основные возможности:
-Чтение и валидация входных файлов (по расширению .sb из директории запуска).
-Упорядочивание данных сотрудников и сохранение их в файлы вида <department_name>.sb, где department_name — название департамента.
-Сортировка сотрудников по заданным параметрам (например: имя, зарплата, порядок сортировки).
-Формирование статистики — вывод в консоль или запись в отдельный файл.
-Обработка ошибок: некорректные или нераспознанные данные записываются в error.log.
-Логирование работы приложения — все события фиксируются в logs/application.log.

--Версия Java: 17

--Система сборки: gradle-8.9

--Библиотеки: 
-Lombok: 1.18.38
-log4j: 2.25.1

--Особенности реализации:
-Полученные выходные файлы (кроме статистики) по умолчанию располагаются в каталоге "output". 
Его можно изменит в package com.korona.file_organizer.config, параметр private Path outputDir = Path.of("output");
-Название департамента и имя работника должны соответстовать следующему регулярному выражению  "^[A-Za-zА-Яа-я\\s]+$"
-Реализовано логирование, логи размещаются в "logs/application.log"

--Инстуркция по запуску:
1) Перейдите в каталог App_Kuzko
2) Соберите fat jar выполнив команду gradlew.bat shadowJar
3) Разместите в App_Kuzko файлы с расширением .sb
4) Запустите приложение выполнив команду java -jar build\libs\app.jar

--Примеры команд для запуска (можно использовать другие согласно требованиям):
 java -jar build\libs\app.jar
 java -jar build\libs\app.jar --sort=name --order=asc --stat
 java -jar build\libs\app.jar -s=salary --order=desc --stat -o=file --path=output/statistics.txt
